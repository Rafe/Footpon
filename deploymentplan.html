Deployment Plan

Testing

To test the application properly, it will be given to a volunteer. This person will need to create an account, log in, gain steps, redeem a coupon, and log out. Then the person will need log in back in, find the coupon they just redeemed, and "use" it. Unfortunately, the coupons are not real, so it cannot be used in stores. Notes will be taken on how the volunteer uses the application and ways to improve it.

Deployment

The application will be released on the Android market where it can be downloaded for free. It will also be available through the website for the application.

Promotion

To promote the application, real coupons will be needed. Participating stores will have flyers advertising their participation and the application. The website will be submitted to search engines so that people will be able to find it in related searches.
